💡 Guia de Uso - Visualizador de Imagens

⌨️ Atalhos de Teclado

ESC Fechar janela
F11 Maximizar/Restaurar
+ ou = Zoom In
- ou _ Zoom Out
0 Reset Zoom (100%)

🖱️ Controles do Mouse

Arrastar Mover imagem
Scroll Zoom In/Out
Duplo Clique Reset Zoom

🔍 Controles de Zoom

Botão + Aumentar Zoom
Botão - Diminuir Zoom

🔄 Controles de Rotação

Botão ↺ Rotacionar 90° Anti-horário
Botão ↻ Rotacionar 90° Horário

⬅️➡️ Navegação

Botão ◀ Imagem Anterior
Botão ▶ Próxima Imagem

💡 Dicas

  • O zoom é sempre centralizado no ponto do mouse
  • Use o scroll para zoom preciso na área desejada
